## Modules Overview
- **CPP01** ‚Äî Basics: stack vs heap, pointers & references, simple classes, and file I/O.
- **CPP02** ‚Äî Ad-hoc polymorphism & class design: operator overloading and Orthodox Canonical Form (Rule of Three).
- **CPP03** ‚Äî Inheritance: base/derived classes, method overriding, and proper ctor/dtor chains.
- **CPP04** ‚Äî Subtype polymorphism: virtual methods, abstract classes, interfaces, and deep copies.
- **CPP05** ‚Äî Exceptions: error handling with `try/catch`, custom exceptions, and RAII mindset.
- **CPP06** ‚Äî Type casts: `static_cast`, `dynamic_cast`, `reinterpret_cast`, `const_cast`, and safe downcasting.
- **CPP07** ‚Äî Templates: function/class templates and generic programming basics.
- **CPP08** ‚Äî STL & iterators: containers, iterators, algorithms, and custom iterable types.
- **CPP09** ‚Äî Algorithms with STL: parsing + problem-solving (BitcoinExchange, RPN, PmergeMe).
